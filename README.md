# Kimbie for Rainbow

A color scheme inspired by [Cold Spring Fault Less Youth][1], the second album by British band Mount Kimbie.

![Animated Screenshot][2]

## Installation

1. Setup [Rainbow][3]
2. Download a `.css` file of choice
2. Include the file in your HTML
